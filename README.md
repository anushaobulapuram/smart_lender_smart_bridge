# Smart Lender AI – AI-Powered Loan Approval Prediction System

Smart Lender AI is a Machine Learning-based web application that predicts whether a bank loan application should be **Approved** or **Rejected**, based on applicant details such as income, credit history, education, and property area. It's built to help banks and financial institutions automate the loan approval process, reduce manual effort, and give applicants a faster, more transparent decision.

🔗 **Live/Repo:** https://github.com/anushaobulapuram/smart_lender_smart_bridge

---

## Features

- **Predict Page** – applicants submit personal & financial details through a guided form
- **Instant Prediction** – a trained ML model returns Approved / Rejected in real time
- **Result Page** – shows Eligibility %, Confidence Score, Risk Level, Reasons, and Recommendations
- **Applicant Profile** – view submitted applicant details
- **History** – track previous prediction records
- **About Page** – project overview and documentation

## How It Works

1. User fills out the loan application form on the **Predict** page
2. Categorical fields (Gender, Married, Education, etc.) are label-encoded
3. Numerical fields (Income, Loan Amount, etc.) are scaled using `StandardScaler`
4. The trained **KNN** model predicts `Loan_Status` and a prediction probability
5. Application logic converts that probability into eligibility %, confidence score, risk level, reasons, and recommendations
6. The **Result** page displays the full explanation

> **Note:** The ML model only predicts Approved/Rejected. Eligibility %, Confidence Score, Risk Level, Reasons, and Recommendations are generated by application logic on top of the model's prediction probability — not by the model itself.

## Dataset

- **Source:** `loan_data.csv` — 614 records
- **Target:** `Loan_Status` (Y = Approved, N = Rejected)
- **Features:** Gender, Married, Dependents, Education, Self_Employed, ApplicantIncome, CoapplicantIncome, LoanAmount, Loan_Amount_Term, Credit_History, Property_Area

## Model

Four classification algorithms were trained and compared using accuracy score, classification report, and confusion matrix:

| Model | Accuracy |
|---|---|
| Decision Tree | 76.42% |
| Random Forest | 83.74% |
| **K-Nearest Neighbors (Deployed)** | **84.55%** |
| XGBoost | Compared during training |

The best-performing model (KNN) was saved and deployed.

## Getting Started

### Prerequisites
- Python 3.9+
- pip

### Installation

```bash
git clone https://github.com/anushaobulapuram/smart_lender_smart_bridge.git
cd smart_lender_smart_bridge
pip install -r requirements.txt
```

### Run the app

```bash
python app.py
```

Then open **http://127.0.0.1:5000** in your browser.

### (Optional) Retrain the model

```bash
python train_model.py
```

This regenerates `models/model.pkl`, `models/encoder.pkl`, and `models/scaler.pkl`.
