# 🏦 Smart Lender – AI-Powered Loan Approval Prediction System

An intelligent Machine Learning-based web application that predicts loan eligibility using applicant information. The system helps banks and financial institutions automate the loan approval process by providing fast, accurate, and data-driven predictions.

---

## 🚀 Features

- 🤖 AI-Based Loan Eligibility Prediction
- 📊 Eligibility Score & Risk Assessment
- 📈 Prediction Analysis
- 💡 AI-Based Recommendations
- 👤 Applicant Profile Management
- 📜 Prediction History
- 📄 PDF Report Generation
- 💰 EMI Calculator
- 📱 Responsive User Interface
- ⚡ Fast and Accurate Predictions

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/anushaobulapuram/smart_lender_smart_bridge.git
```

### Move into the Project

```bash
cd smart_lender_smart_bridge
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Train the Model (if required)

```bash
python train_model.py
```

### Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
