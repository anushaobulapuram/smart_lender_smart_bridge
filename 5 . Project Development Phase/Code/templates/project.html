Project Overview

Smart Lender AI is an advanced decision support system designed specifically for banking institutions to assess credit risk and evaluate loan eligibility in real-time. By leveraging supervised machine learning models trained on structural credit records, the system automates preliminary reviews, reduces human review bottlenecks, and maintains consistent risk management standards.

Machine Learning Workflow

Step-by-step pipeline from applicant files submission to credit risk prediction result.

01

Dataset Collection

Ingests historical loan portfolios containing 614 unique credit records.

02

Data Preprocessing

Cleans data rows, handles missing null values, and normalizes feature distributions.

03

Feature Encoding

Applies label encoding to categorical variables like Gender, Married, Dependents, and Property Area.

04

Model Training

Trains and benchmarks multiple algorithms (Decision Trees, Random Forest, KNN, XGBoost) to select the optimal model.

05

Prediction Execution

Processes submitted applicant parameters against the best-performing model to evaluate risk levels in real-time.

06

Loan Eligibility Result

Displays the final decision (Approved/Rejected) and caches to local terminal storage logs.
